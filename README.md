## Commands

### Install
``` yarn install ```

### Start watch
``` gulp serve ```
or
``` yarn start ```

### Start build
``` gulp ```
or
``` yarn build ```

##### Node >= v.6.11
##### Yarn >= v1.0.1
##### Serve port: [3000](http://localhost:3000/)
